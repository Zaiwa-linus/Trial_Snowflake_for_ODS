# e-Stat データ取得スキル

テーマに沿った統計データをe-Statで探し、都道府県（またはそれより細かい粒度）での集計が可能なものをユーザーに提案してダウンロードする。

## 実行フロー

### Step 1: 検索

テーマに関連するキーワードで統計表を検索する。複数キーワードで検索してよい。

```bash
uv run python .claude/skills/estat/scripts/search_stats.py "キーワード" --limit 30
uv run python .claude/skills/estat/scripts/search_stats.py "キーワード" --field XX --limit 30
```

統計分野コード（`--field`）:
| コード | 分野 |
|--------|------|
| 01 | 人口・世帯 |
| 02 | 自然環境 |
| 03 | 経済基盤構造 |
| 04 | 政府・財政・金融 |
| 05 | 農林水産業 |
| 06 | 鉱工業 |
| 07 | 商業・サービス業 |
| 08 | 企業活動 |
| 09 | 住宅・土地・建設 |
| 10 | エネルギー・水 |
| 11 | 運輸・観光 |
| 12 | 情報通信・科学技術 |
| 13 | 教育・文化・スポーツ・生活 |
| 14 | 行財政 |
| 15 | 司法・安全・環境 |
| 16 | 社会保障・衛生 |
| 17 | 国際 |
| 18 | その他 |

### Step 2: 都道府県粒度の確認

検索結果の中から有望な統計表を選び、メタ情報で地域分類を確認する。

```bash
uv run python .claude/skills/estat/scripts/get_meta.py {統計表ID} --json
```

**都道府県粒度と判定する条件**（以下のいずれかを満たす）:
- `area` または `cat` 系の分類に都道府県コード（`01000`〜`47000` 形式、または2桁の `01`〜`47`）が含まれる
- 分類名に「都道府県」「地域」「全国・都道府県」などが含まれる
- 市区町村コード（5桁）が含まれる場合はより細かい粒度として OK

**除外するもの**:
- 全国集計のみ（地域分類が「全国」1件だけ）
- 地方ブロック（東北・関東など）のみで都道府県がない

### Step 3: ユーザーへの提案

都道府県粒度が確認できた統計表を一覧でユーザーに提示する。

提示フォーマット:
```
以下の統計表が見つかりました。ダウンロードする帳票を選んでください。

1. **{統計名} - {タイトル}**
   - 統計表ID: {ID}
   - 提供機関: {機関名}
   - 周期: {周期}
   - データ件数: {件数}
   - 地域粒度: {都道府県 or 市区町村}
   - 概要: {何のデータか一言}

2. ...

番号で選択するか、「すべて」「なし」と回答してください。
```

### Step 4: ダウンロード

ユーザーの許諾を得た統計表のみダウンロードする。

```bash
uv run python .claude/skills/estat/scripts/download_data.py {統計表ID}
```

- 保存先: `data/{統計表ID}/{統計表ID}.csv` および `data/{統計表ID}/column_summary.yml`
- ダウンロード完了後、保存先パスをユーザーに報告する

## 注意事項

- 許諾なしにダウンロードしない
- 件数が極端に多い（100万件超）場合はユーザーに事前に伝える
- 同じ統計表IDが既に `data/` に存在する場合は上書きになる旨を伝える
